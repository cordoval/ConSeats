ConSeats
========

Demo application developed during "Advanced PHP Training" at ConFoo 2012.


Requirements
------------

* Events
  * have a date
  * have a name
  * have a list (or map) of seats

* Seats
  * are numbered
  * can be available
  * can be reserved

* Persons
  * can reserve one or more seats
  * can cancel a reservation

* List of seats
* Reservation
* Cancellation
